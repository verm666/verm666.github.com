HISTTIMEFORMAT

written on Monday, October 3, 2011

  • Q: Как в history вывести время выполнения команды?
  • A: Выставить значение переменной HISTTIMEFORMAT (в формате strftime)
# export HISTTIMEFORMAT="+%Y/%m/%d - %H:%M:%S % "
# history 3
    44  2011/10/03 - 16:27:55 % ls
    45  2011/10/03 - 16:27:59 % export HISTTIMEFORMAT="%Y/%m/%d - %H:%M:%S % "
    46  2011/10/03 - 16:28:01 % history 3
